<h2 align="center">Установка и использование</h2>
Установка зависимостей
```bash
npm i
```
### Разработка
Для запуска в режиме разработки используется команда:
```
npm run gulp
```

Сборка проекта
```bash
npm run build
```

Запуск
```bash
npm start
```
