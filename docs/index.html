Онлайн-чат [utp2018-1-chat]

Описание

Клиент-серверное приложение «Онлайн-чат», реализованное в рамках учебной практики, студентами первого курса кафедры ИУ9 МГТУ им. Баумана.

Функционал

  • Обмен текстовыми сообщениями.
  • Групповые чаты (беседы) на несколько человек.
  • Поддержка Emoji.
  • Поддержка синтаксиса Markdown.
  • Поиск по сообщениям.

Используемые инструменты и технологии:

Клентская часть:

  • HTML5
  • SASS (препроцессор CSS3)
  • JavaScript

Серверная часть:

  • Node.js (веб-сервер)
  • RxDB [LevelDB] (база данных)

Дополнительно:

  • Git (для командной разработки)
  • Gulp (сборка проекта)
  • GitBook (написание документации)
